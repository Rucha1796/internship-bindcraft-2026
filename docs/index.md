# Designing Protein Binders with AI

**Computational Protein Design Internship — Summer 2026**

Welcome! Over the next 4–6 weeks, you will go from understanding what a protein is to designing your own protein binders against a real cancer target using state-of-the-art AI tools — the same tools used in research labs and biotech companies around the world.

By the end of this internship, you will have:

- A solid understanding of how proteins work, fold, and interact
- Hands-on experience with AlphaFold2, the tool that revolutionized biology in 2021
- Designed real protein binders against PD-L1, a key cancer immunotherapy target
- Analyzed and selected designs to recommend for wet lab testing
- (6-week students) Applied this entire workflow independently to a new target

## How to Use This Wiki

- **Read first, then do.** Each module page has background reading followed by a hands-on activity or notebook.
- **Notebooks are pre-written.** You don't need to write code from scratch — run the cells, change the highlighted parameters, and observe what changes.
- **Ask questions.** Nothing here is meant to be obvious. If something doesn't make sense, ask your mentor.
- **The [Glossary](glossary.md)** has plain-English definitions for every technical term used in this wiki.
